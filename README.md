# contact-app
 Import contacts feature.

 First Clone the Repository and Open it in VS Code or any Code Editor

 To Deploy Backend on Local Machine :
- cd backend
- npm i
- node server.js

To Deploy FrontEnd on Local Machine :
-cd Frontend
-npm i
-npm run start
